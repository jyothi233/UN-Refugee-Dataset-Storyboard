# UN-Refugee-Dataset-Storyboard
The "Refugees Storyboard Project" uses Tableau to analyze global refugee trends and demographics, offering insights into displacement patterns and host countries' responses .
Key visualizations include:

1. Refugee Population Trends Over Time :
 - Charts display the growth of refugees under UNHCR's mandate from 1950 to the present, highlighting significant peaks in displacement.

2. Top Host Countries:
 - Visual breakdown of countries providing asylum, emphasizing key contributors like the United States, Germany, and Sweden.

3. Geographical Distribution:
 - Interactive maps showcase the distribution of refugee populations across regions and countries, revealing areas with the highest impact.

4. Demographics:
 - Analysis of refugee origins, host countries, and demographic insights, filtered by metrics like asylum-seekers and population size.

The project highlights the use of Tableau features such as filters, calculated fields, and interactivity to address humanitarian issues. It is a comprehensive visualization of displacement data, enabling stakeholders to understand and strategize on refugee support.
